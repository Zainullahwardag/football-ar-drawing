# Football AR Drawing

A modern UI/UX design for an Augmented Reality football drawing mobile application.

## Project Overview

Football AR Drawing helps users trace football sketches using Augmented Reality. This project includes a complete onboarding flow, App Store screenshots, and localized marketing assets for multiple countries.

## Responsibilities

- UI Design
- UX Design
- App Store Screenshot Design
- Screenshot Localization
- Marketing Asset Design

## Features

- AR Drawing
- Camera Projection
- Easy Tracing
- Professional Sketches

## Tools Used

- Adobe Photoshop
- Figma
- Canva

## Localizations

- English
- Spanish
- Portuguese (Brazil)
- French
